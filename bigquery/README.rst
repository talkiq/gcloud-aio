(Asyncio OR Threadsafe) Python Client for Google Cloud BigQuery
===============================================================

    This is a shared codebase for ``gcloud-aio-bigquery`` and
    ``gcloud-rest-bigquery``

|pypi| |pythons|

Installation
------------

.. code-block:: console

    $ pip install --upgrade gcloud-{aio,rest}-bigquery
