gcloud.rest.bigquery

This library implements various methods for working with the Google Bigquery APIs.

Installation

$ pip install --upgrade gcloud-rest-bigquery

Usage

We're still working on documentation -- for now, you can use the smoke test as an example.

Emulators

For testing purposes, you may want to use gcloud-rest-bigquery along with a local emulator. Setting the $BIGQUERY_EMULATOR_HOST environment variable to the address of your emulator should be enough to do the trick.
