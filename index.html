(Asyncio OR Threadsafe) Google Cloud Client Libraries for Python¶

These docs describe the shared codebase for gcloud-aio-* and gcloud-rest-*. Both of them are HTTP implementations of the Google Cloud client libraries: the former has been built to with asyncio (using the aiohttp library) and the latter is a threadsafe requests-based implementation.
